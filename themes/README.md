# Themes

Pi TUI themes adapted from well-known editor color schemes.

## Kanagawa

Based on [Kanagawa.nvim](https://github.com/rebelot/kanagawa.nvim) by rebelot,
inspired by Hokusai’s *The Great Wave off Kanagawa*.

| File | Variant | Feel |
| --- | --- | --- |
| `kanagawa-wave.json` | Wave | Dark, deep sumi inks with wave aquas and spring greens |
| `kanagawa-lotus.json` | Lotus | Light, warm paper tones with soft lotus inks |

## Rosé Pine

Based on the [Rosé Pine](https://rosepinetheme.com/) palette.

| File | Variant | Feel |
| --- | --- | --- |
| `rose-pine.json` | Main | Dark, muted purples with rose, foam, and gold accents |
| `rose-pine-dawn.json` | Dawn | Light, warm paper base with the same accent family |

## Notes

The `thinking*` colors (`thinkingOff` through `thinkingMax`) are intentionally
flat within each theme: every thinking level uses the same color. Pi's default
themes ramp those levels; these ports keep the statusline quieter and closer to
the source palettes.

## Usage

Select a theme in Pi with `/theme`, or set it in settings. After installing or
updating this package, run `/reload` so newly added themes appear.
